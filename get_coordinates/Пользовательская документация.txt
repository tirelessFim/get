*****Руководство пользователя*****

- Запустите файл "main.py"

	Появится стартовое окно с выбором действий
	___________________________________________
      		Программа для поиска координат.
	--------------------------------------------
	1. Команда "start" - для начала работы
	2. Команда "quit" - для выхода из программы
	____________________________________________

В полее ввода допускается ввод только двух команд "start" и "quit".
Все символы в нижнем регистре. Остальные варианты набора не допускаются. 


1. Команда "start":

1.1. Запуск программы. 
	
	Для поиска кординат по адресам необходимо предварительно ввести следующие данные:
		- token(Секретный ключ)
		- API ключ
		- язык (ru/en)
	
	!Выжно: проверяйте корректность ввода данных. В случае ошибок ввода, программа не будет работаь корректно.
	Язык запроса стоит по умолчанию - ru, если не выбрать вариант en.

1.2. Поиск адреса.
	После настройки программы предлагается ввести адрес для поиска координат. 
	Для более точного результата рекомендуется использовать шаблон для ввода адреса:
		г "Город" ул "улица" д "Номер дома", пример - (г Новосибирск ул Ленина д 3)
	
	После ввода адреса выдает список из вариантов существующих вариантов в базе.
	Для уточнения запроса, выберите номер предложенного варианта из списка.
	
	!Важно: Для правельной работы программы вводить только целое число или команду "quit" для выхода из программы.
		Целое число дожно быть не больше количества выпавших вариантов и не меньше единицы.

1.3. Результат поиска.
	
	В результате поиска указывается выбранный вариант из списка вариантов.
	Указывается Широта и Долгота 

После вывода результата процесс с поиском адреса можно повторить, вы вернетесь к процессу 1.2.

2. Команда "quit"
	
	Команда quit прерывает работу программы на любом этам ввода пользователя данных, 
	кроме ввода в строке с выбором языка.
